**Для запуска приложения необходимо установить docker и преключиться на linux контейнеры.**

- запустить командный файл start.cmd который скачает и настроит контейнер RabbitMQ. 

- потом можно запускать приложенния Receiver.exe и Sender.exe из папки deploy. После того как контейнер RabbitMQ запустится они автоматически подключится к нему.

Логи отсылаемых сообщений будут сохранены в файле senderlog.db3, а логи принимаемых сообщений будут сохранены в файле receiverlog.db3. 
Просмотреть содержимое логов мождно с помощью утилиты DB Browser for SQLite скачать ее можно по адресу https://sqlitebrowser.org/

- после завершения тестирования нужно запустить файл stop.cmd который удалит созданный контейнер docker

Исходные файлы находятся в папке sources
